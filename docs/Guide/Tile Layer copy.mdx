# Capture Map

```jsx
<div>
  <MapContainer
    center={[126.840884, 35.190816]}
    width="100%"
    height="500px"
  >
    <CaptureMap onCaptured={(img)=>{console.log(img)}}>
  </MapContainer>
</div>
```

| property   | type | description                                                                         |
| ---------- | ---- | ----------------------------------------------------------------------------------- |
| onCaptured | func | map rendering이 마무리 되면 onCaptured를 통해 이미지 바이너리를 생성할 수 있습니다. |
