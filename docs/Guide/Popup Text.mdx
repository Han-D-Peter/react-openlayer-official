# Popup Text

## How to use

```jsx live
<div>
  <Map center={[126.840884, 35.190816]} width="100%" height="500px">
    <CustomMarker center={[126.840884, 35.190816]}>
      <InnerText isPopup>Marker</InnerText>
    </CustomMarker>
  </Map>
</div>
```
