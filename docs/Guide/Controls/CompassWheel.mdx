# CompassWheel

```jsx live
<div>
  <MapContainer center={[126.841784, 35.191406]} width="100%" height="500px">
    <CompassWheel />
  </MapContainer>
</div>
```

- `CompassWheel` 을 통해 현재 지도의 방위각을 파악할 수 있습니다.
- 설정을 원하는 방위각을 입력해 특정 각으로 변경이 가능합니다.
- 낫침반을 마우스로 조장하면 방위각을 조정할 수 있습니다.
