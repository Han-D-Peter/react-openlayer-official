# Image Overay

```jsx live
<div>
  <MapContainer center={[126.840884, 35.190816]} width="100%" height="500px">
    <ImageOverlay
      imageUrl="/img/docusaurus-social-card.jpg"
      bounds={[
        [126.83984, 35.190016],
        [126.841984, 35.191516],
      ]}
    />
  </MapContainer>
</div>
```

| property | type                              | description                        |
| -------- | --------------------------------- | ---------------------------------- |
| bounds   | [[number,number],[number,number]] | 왼쪽 아래 경위도, 오른쪽 위 경위도 |
